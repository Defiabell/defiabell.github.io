搭建个人博客

Posted by on Sat, Dec 2, 2023

起因

老大一直有关注我的个人 IP 建设。作为一个技术方向的后端工程师，以架构师为目标的职业路线还是要尽早建设自己的个人 IP。

工作了五年多，焦虑在逐渐加重，终于在今天（2024.03.29）再次捡起个人博客。

遇到的问题

问题1：hugo server 命令启动的服务显示 page not found。

查询发现说没有配置 theme，但我确实有在 config.toml 配置了 theme 参数。然后使用命令 hugo server -t Blogpaper 启动后发现是有界面的。

问题2：hugo 命令执行后没有生成对应的静态页面。

后来意识到我的 hugo 是很早之前就安装了，到现在已有半年之久了，版本更新可能带了一些不一样的变化，所以尝试更新到最新版试试。

升级 hugo 版本后重新执行 hugo 命令，但 content 目录下创建的文件还是没有生成。 经过多次尝试，发现居然是命名问题，默认content文件夹下生成的 index.md 改为 _index.md，然后重新生成一下就好了
