搭建个人博客

遇到的问题：hugo server 命令启动的服务显示 page not found。查询发现说没有配置 theme，但我确实有在 config.toml 配置了 theme 参数。然后使用命令 hugo server -t Blogpaper 启动后发现是有界面的。 问题2: hugo 命令执行后没有生成对应的静态页面。

后来意识到我的 hugo 是很早之前就安装了，到现在已有半年之久了，版本更新可能带了一些不一样的变化，所以尝试更新到最新版试试。

升级 hugo 版本后重新执行 hugo 命令，但 content 目录下创建的文件还是没有生成。 经过多次尝试，发现居然是命名问题，默认content文件夹下生成的 index.md 该名为 _index.md，然后重新生成一下就好了
