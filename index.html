《周处除三害》观后感

April 5, 2024

本文对影片内容有很多描述，介意剧透请勿阅读 《周处除三害》，儿时的回忆突然袭击我的脑海。刚开始看到这个名字的时候还没有意识到是小时候曾经看过的一个故事。好像是小学语文课本上的，当时看还很热血沸腾，最后看到周处自己意识到自己是人们口中的第三害而痛苦时，还很为周处感到不值。 《周处除三害》这个电影讲的是一个叫陈桂林的黑社会小弟(应该是小弟)，枪杀一个黑社会老大后逃跑。四年后发现自己是肺癌晚期，自首时发现自己几乎被大众，被警察所遗忘了，而且还有两个排在前面的嫌疑犯，为了能让人们记住自己，决定干掉另外两个嫌疑犯的故事。 这个电影算半个爽剧，但看完却让人很有感触。可能每个观影者都因为个人经历及思想意识的差别而有不同的感受，下文内容纯属个人的思考及观点。如有冒犯请勿介意。 首先开始，陈桂林出场枪杀大人物，然后逃脱警察的追捕。在枪杀过程中，从其他黑社会小弟的反应（蹲下躲闪不敢动）也衬托出了陈桂林超众的能力（不是任何人都能成为搏杀蛟龙与恶虎的周处）。紧接着是与刑警的跑酷与打斗场景，最终陈桂林逃脱。 时间来到四年后。陈桂林的奶奶去世了。陈桂林见奶奶的最后一面已经是四年前，虽然给奶奶邮寄了一百万，但奶奶一分没花，只给自己买了一个粉色的手表。 这里奶奶以及第二害的嫌疑犯的母亲都是一个人在孤独中去世。嫌疑犯不敢与亲人（哪怕是唯一的亲人）接触，甚至不敢打上一通电话。 陈桂林的医生告诉他也已经是第四期肺癌（没救），劝他去自首。奶奶的去世和肺癌的双重打击下，陈桂林决定自首。自首时正好遇到一起钞票散落被市民捡走后送到警局的事故。警员在忙乱中没有认出陈桂林是重要嫌疑犯。陈桂林内心也发生了一些变化，到他看到通缉令上的另外两个人后，决定自首前先解决掉另外两个危害。 人们需要被关注，被忽视是可怕的 人类社会是集群社会，每个人都想要被关注。相信很多人都有过要么扬名立万，要么臭名远扬的想法。 被关注是一种精神上的需要，无论是向上社交还是向下社交。一个优秀的推销员，一个亲和的 Leader，一个完美的伴侣都会让其他人有一种被关注，被重视的感觉。 第一个危害是个毒贩。一个游离在道德法律外的毒贩，杀人仿佛家常便饭。 这一段属于纯暴力击杀。靠自身硬实力解决了第一个嫌疑犯。同时救出了被毒贩控制的程小美。 小美的母亲是被毒贩从另一个坏人那里解救（杀了另一个坏人）出来的。所以为毒贩扛下了贩毒的罪名进了监狱。小美第一次被提及是十岁的时候为毒贩取药。毒贩对小美的禽兽行为，可以看出小美对毒贩对无声的反抗，但又逃不出的绝望，以及对陈桂林能解决毒贩的希望。 小美潜意识里以为从一个魔掌到另一个魔掌到时候，陈桂林说：你自由了。 社会上一定有很多像小美一样处在黑暗中的人。 无法想象还有哪些闻所未闻的黑暗。 第二个嫌疑人是一个改头换面的静心舍尊者（传销头头）。陈桂林到来后被尊者洗脑感化，而后发现了尊者丧尽天良的传销本质后干掉了尊者，以及一些几乎彻底麻木的成员 传销的故事很恐怖。是那种披着天使外衣的魔鬼般的可怕。 当紧急事情发生在自己身上，很容易被人操纵，仿佛溺水的人会拼命抓住任何一个救命稻草。 所思所感 如果陈桂林没有得知自己是肺癌晚期，他会怎样？ 可能会继续过着隐姓埋名的躲藏生活。电影中对陈桂林的刻画很真实，没有渲染他是一个纯粹的恶人，也没有刻画他纯粹的善良。他像多数人一样惜命，自首前九次向关公求签。 如果他奶奶还在，他会怎样？ 如果奶奶还在的话，得知自己肺癌晚期，陈桂林可能会去自首，但可能不会去杀另两个嫌疑人。是奶奶的去世让他无所牵挂。 为啥他能找到嫌疑犯，而警察找不到？ 很多时候‘好人’ 会受到各种限制，例如陈桂林可以绑架医生的孩子进行要挟，而警察做不到。 传销洗脑是真的可以改变一个人 有些事情你见到的只是别人想让你见到的。 我们日常接收到的信息是不是也被有意控制的，我们是不是也被打下了思想钢印「出自《三体》」。 最后陈桂林后悔了吗？ 最后陈桂林知道了自己并没有肺癌，是医生为了让他自首而骗他的。肯定会有悔恨。悔恨自己人生岔路口的选择，悔恨自己最后见奶奶已是四年前。 刮胡子时为啥哭了？ 是我我也哭。这种在绝境（必死）中感受到温情是最能撩动心扉的。 没有纯粹的好人和坏人。我们对于好人坏人的评判标准是不是被洗脑的结果？ 人不为己，天诛地灭；还是但行好事莫问前程 看历史，看传记，发现很多帝皇、名人、富豪都有自己的局限，他们会做出很多不符合道德法律标准的事情。为啥大多数人所奉行的道德规范在名人那里很灵活？为啥靠道德规范很难发家致富出人头地？ 如果一个人有了超出常人的能力，会是什么样？ 无论是漫威英雄还是各国神话传说，超能力人群都会有一种舍身取义的精神。可以不顾自己生死去救一个不认识的人甚至一只猫。明明可以不受束缚打破伦理却在被人误解被人唾弃的时候咬牙坚持正义。 这些神话传说，文学，电影等是不是就是一种洗脑行为？为了塑造人们的价值观。出发点也许是好的，毕竟人生活在社会中，社会是个集群…

Poor Charlie's Almanack

December 10, 2023

《穷查理宝典》

缅怀 查理·芒格。 之前偶有听起这本书，但直到老先生去世，才开始真正捧起这本书。 读完这本书后，留下的最大的印象是 芒格 对阅读的热枕。 作为一个互联网时代的普通人，感觉每天都在接收海量信息，但基本都是些无用的信息，甚至偶尔感觉到在被定向投喂。 思考好像越来越少，有时候会被一句话或一个视频所震感，然后产生短暂的热血与反思。过些时日又被淹没在唾手可得的信息潮中。 有时候也会发现一些令人大开眼界的观点。也许我们日常所能看到的都是某些人想让我们看的，甚至我们的思想，我们的价值观都在被日常生活中的信息所塑造。在没有外力的干扰下，我们很难发现自己生活在《楚门的世界》。 世上大多是赢者通吃的游戏，有钱人更容易有钱，当权者更容易获取资源。 普通人也有升官发财的希望，但千军万马过独木桥。普通人想实现阶级越迁很难，甚至要经过几辈的努力。 思考和阅读同样重要。阅读可以提高见识，可以塑造价值观。在充斥海量信息的互联网上，阅读门槛变低了，但如何寻找有价值的读物正变得越来越难。几乎所有信息的产生都伴随着作者的主观意识，阅读中要加入自己的思考。 芒格 一生都在坚持实践自己的原则 不知为何，现在人们好像很难说出自己的热爱是什么？自己追求的人生是什么样的？生活的琐碎，生存的压力，物欲的诱惑，情感的羁绊… 种种原因导致人们无法突破现有的人生，无法做出想要的改变。当然更多是不知道自己要追求什么样的人生的迷茫。

搭建个人博客

December 2, 2023

起因 老大一直有关注我的个人 IP 建设。作为一个技术方向的后端工程师，以架构师为目标的职业路线还是要尽早建设自己的个人 IP。 工作了五年多，焦虑在逐渐加重，终于在今天（2024.03.29）再次捡起个人博客。 遇到的问题 问题1：hugo server 命令启动的服务显示 page not found。 查询发现说没有配置 theme，但我确实有在 config.toml 配置了 theme 参数。然后使用命令 hugo server -t Blogpaper 启动后发现是有界面的。 问题2：hugo 命令执行后没有生成对应的静态页面。 后来意识到我的 hugo 是很早之前就安装了，到现在已有半年之久了，版本更新可能带了一些不一样的变化，所以尝试更新到最新版试试。 升级 hugo 版本后重新执行 hugo 命令，但 content 目录下创建的文件还是没有生成。 经过多次尝试，发现居然是命名问题，默认content文件夹下生成的 index.md 改为 _index.md，然后重新生成一下就好了
