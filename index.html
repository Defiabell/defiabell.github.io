Poor Charlie's Almanack

December 10, 2023

《穷查理宝典》

穷查理宝典 缅怀 查理·芒格。 之前偶有听起这本书，但直到老先生去世，才开始真正捧起这本书。 读完这本书后，留下的最大的印象是 芒格 对阅读的热枕。 作为一个互联网时代的普通人，感觉每天都在接收海量信息，但基本都是些无用的信息，甚至偶尔感觉到在被定向投喂。 思考好像越来越少，有时候会被一句话或一个视频所震感，然后产生短暂的热血与反思。过些时日又被淹没在唾手可得的信息潮中。 有时候也会发现一些令人大开眼界的观点。也许我们日常所能看到的都是某些人想让我们看的，甚至我们的思想，我们的价值观都在被日常生活中的信息所塑造。在没有外力的干扰下，我们很难发现自己生活在《楚门的世界》。 世上大多是赢者通吃的游戏，有钱人更容易有钱，当权者更容易获取资源。 普通人也有升官发财的希望，但千军万马过独木桥。普通人想实现阶级越迁很难，甚至要经过几辈的努力。 思考和阅读同样重要。阅读可以提高见识，可以塑造价值观。在充斥海量信息的互联网上，阅读门槛变低了，但如何寻找有价值的读物正变得越来越难。几乎所有信息的产生都伴随着作者的主观意识，阅读中要加入自己的思考。 芒格 一生都在坚持实践自己的原则 不知为何，现在人们好像很难说出自己的热爱是什么？自己追求的人生是什么样的？生活的琐碎，生存的压力，物欲的诱惑，情感的羁绊… 种种原因导致人们无法突破现有的人生，无法做出想要的改变。当然更多是不知道自己要追求什么样的人生的迷茫。

搭建个人博客

December 2, 2023

起因 老大一直有关注我的个人 IP 建设。作为一个技术方向的后端工程师，以架构师为目标的职业路线还是要尽早建设自己的个人 IP。 工作了五年多，焦虑在逐渐加重，终于在今天（2024.03.29）再次捡起个人博客。 遇到的问题 问题1：hugo server 命令启动的服务显示 page not found。 查询发现说没有配置 theme，但我确实有在 config.toml 配置了 theme 参数。然后使用命令 hugo server -t Blogpaper 启动后发现是有界面的。 问题2：hugo 命令执行后没有生成对应的静态页面。 后来意识到我的 hugo 是很早之前就安装了，到现在已有半年之久了，版本更新可能带了一些不一样的变化，所以尝试更新到最新版试试。 升级 hugo 版本后重新执行 hugo 命令，但 content 目录下创建的文件还是没有生成。 经过多次尝试，发现居然是命名问题，默认content文件夹下生成的 index.md 改为 _index.md，然后重新生成一下就好了
